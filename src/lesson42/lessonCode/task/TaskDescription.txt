Разработать систему для управления оценками студентов,
которая позволит хранить информацию о студентах, их предметах и оценках.
Система должна поддерживать добавление и удаление студентов,
добавление оценок по различным предметам, а также возможность получения
статистики по оценкам.

Основные требования:

    Хранение информации о студентах:
    Используйте коллекцию для хранения информации о студентах.
    Каждый студент должен иметь уникальный идентификатор (ID), имя и фамилию.

    Уникальный набор предметов: Используйте коллекцию для хранения списка
    предметов, которые изучают студенты.
    Необходимо автоматически поддерживать предметы в отсортированном
    порядке без дубликатов.

    Хранение оценок: Используйте коллекцию для ассоциации каждого
    студента (ID студента) с другой коллекцией, которая ассоциирует
    предмет с оценками этого студента по данному предмету.
    Для оценок предусмотрите возможность хранения нескольких значений
    (например, в виде другой коллекции).

Задачи для выполнения:

    Добавление и удаление студентов:
    Разработайте методы для добавления новых студентов в систему и
    удаления студентов по ID.

    Добавление предметов:
    Разработайте метод для добавления нового предмета в систему,
    который автоматически добавляется в коллекцию.

    Добавление оценок:
    Реализуйте метод для добавления оценок студентам по конкретному предмету.
    Убедитесь, что студент и предмет уже существуют в системе.

    Получение статистики:
    Создайте методы для расчета средней оценки по каждому предмету для
    студента, а также для определения студента с наивысшим средним баллом
    по всем предметам.
